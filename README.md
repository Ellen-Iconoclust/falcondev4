# Ellen Portfolio

A creative developer portfolio featuring immersive 3D graphics, smooth animations, and modern web technologies.

## 🚀 Tech Stack

- React 18
- Three.js for 3D graphics
- GSAP for animations
- Lenis for smooth scrolling
- TailwindCSS for styling
- Deployed on Vercel

## ✨ Features

- Interactive 3D wireframe background
- Smooth scroll animations
- Text carousel with character animations
- Parallax scrolling effects
- Project reveal animations
- Responsive design
- Glass morphism UI elements
